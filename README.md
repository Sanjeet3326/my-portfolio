# My Portfolio

A modern cybersecurity-themed personal portfolio website built with HTML, CSS, and JavaScript.

## About

This portfolio presents:
- Profile and introduction
- Education background
- Cybersecurity and programming skills
- Certificates with visual proof
- Featured projects
- Resume download
- Contact and social links

## Run Locally

No build step is required.

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Personalization Guide

To customize this portfolio for future updates:
- Update profile text in `index.html`
- Update styles and theme in `style.css`
- Update interactions and animations in `script.js`
- Replace profile and project images in `images/`
- Replace certificate screenshots in `screenshots/`
- Replace resume PDF in the root folder
